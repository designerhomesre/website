Executive Summary

Designer Homes Real Estate Services requires a comprehensive business management platform that integrates seamlessly with the public website. This platform will streamline operations from initial lead intake through final report delivery and payment collection—all while maintaining a target operating cost as close to $0/month as possible.

What's Being Built

A complete, integrated system consisting of:

  • Public Website — Static HTML site hosted free on Netlify, optimized for lead generation and lender inquiries
  • Admin Dashboard — Self-contained HTML/JavaScript application for managing assignments, invoicing, and reporting
  • Client Portal — Authenticated portal for clients to access reports and make payments
  • Payment Processing — Stripe integration for professional payment collection with zero monthly fees
  • Email Automation — Zapier + Google Workspace for workflow automation

Core Workflow

The platform automates the complete business lifecycle:

Lead Intake (Web Form) ↓ Quote & Assignment Creation ↓ Inspection Scheduling & Completion ↓ Report Writing & Finalization ↓ Invoice Generation & Payment Link ↓ Client Payment Via Stripe ↓ Report Release to Client Portal ↓ Review Request (7-day automation)

Budget Target

Baseline Monthly Cost: ~$7/month (Google Workspace for business email)

Variable Costs: Stripe transaction fees (2.9% + $0.30 per card transaction, 0.8% for ACH transfers)

All other infrastructure is provided at no cost through free tiers of Netlify, Supabase, HubSpot, Zapier, and Google Analytics.

Scalability Path

The architecture is designed to scale from MVP to enterprise without architecture changes. When volume justifies it, the system can be upgraded to Supabase's paid tier (unlimited database and file storage) or integrated with professional accounting software (Wave, QuickBooks).

Payment Processor Recommendation

Selecting the right payment processor is critical for client convenience, professional operations, and cost efficiency. Three options have been evaluated.

Option 1: Stripe (RECOMMENDED)

Primary Recommendation Stripe is the best choice for growth and professionalism, with zero monthly fees and transaction-only costs.

Pricing

  • Card Transactions: 2.9% + $0.30 per transaction
  • ACH/Bank Transfers: 0.8% (capped at $5 per transaction)
  • Monthly Fee: $0

Key Features

  • Stripe Checkout: Hosted payment page—no PCI compliance burden on you
  • Professional Payment Links: Send branded payment link via email to clients
  • Invoice Generation: Built-in invoice creation and numbering
  • Recurring Billing: Support for subscription-based services in future
  • Mobile-Friendly: Works perfectly on phones and tablets
  • Webhooks: Automatic payment confirmation notifications
  • Detailed Reporting: Transaction history, dispute tracking, settlement info
  • Zapier Integration: Native integration for automation

Example: $3,000 Appraisal Report

Payment Method Stripe Fee Amount You Receive
Credit Card $0.97 (2.9% + $0.30) $2,999.03
Bank Transfer (ACH) $5.00 (0.8% capped) $2,995.00

Option 2: Square Invoices (Alternative)

Pricing

  • Card Transactions: 2.9% + $0.30
  • ACH/Bank Transfers: 1% (minimum $1)
  • Monthly Fee: $0

Comparison to Stripe

  • Slightly higher ACH fees (1% vs. 0.8%)
  • Excellent invoice reminders (automatic follow-ups)
  • Slightly simpler interface
  • No Zapier integration (less automation potential)

Verdict: Stripe is superior due to lower ACH costs, Zapier integration, and more professional appearance.

Option 3: Zelle (Current)

Pricing

  • All Transactions: $0
  • Monthly Fee: $0

Pros

  • Zero fees (best for you)
  • Instant bank-to-bank transfer

Cons

  • No invoice generation or professional appearance
  • No payment tracking or automation
  • No mobile checkout experience
  • No Zapier integration
  • Clients must manually input payment details
  • No transaction history or reporting

Recommended Strategy

Primary: Use Stripe as primary payment processor (professional, automated, trackable)

Secondary: Keep Zelle as fallback option for clients who prefer direct bank transfers

Benefit: Stripe captures 90% of clients (card payers + convenience seekers), Zelle handles the remaining 10% (privacy-conscious clients)

Implementation Steps

  1. Create Stripe account (5-minute signup at stripe.com)
  2. Verify business information
  3. Connect bank account for payouts
  4. Generate first payment link from admin dashboard
  5. Send payment link to client via email
  6. Configure Zapier webhook to notify you of payments

Mileage Tracker Architecture

Mileage is often the largest deductible business expense for appraisers. The IRS allows $0.70/mile for 2025–2026 (verify annually). Proper tracking is essential for maximizing your tax deduction and defending against audits.

IRS Requirements

To claim mileage deductions, you must be able to document:

  • Date of trip
  • Business Purpose (e.g., "Appraisal inspection of 123 Main St, including 4 comparable property inspections")
  • Start and End Locations (e.g., office → subject property → comp 1 → comp 2 → back to office)
  • Total Miles

Mileage Logging Workflow

Admin dashboard provides simple workflow:

  1. Open Assignment — Click "Log Mileage" from assignment detail view
  2. Enter Trip Date — Pre-filled with today's date
  3. Subject Property Address — Automatically linked to assignment
  4. Log Comparable Properties — List each comp property visited
    • Address field
    • Checkbox: "Visited this property"
  5. Calculate Mileage — System estimates route:
    • MVP Formula: Configurable base distance (office to subject) + increment per comp visited
    • Default: 15 miles (office to subject) + 5 miles per comp property
    • Example: 1 subject + 4 comps = 15 + (4 × 5) = 35 miles
  6. Manual Override — Override calculated miles if you use actual GPS data
  7. Save Trip — Trip recorded and linked to assignment

Future Enhancement: Google Maps Integration

When you're ready to improve accuracy, integrate Google Maps Distance Matrix API:

  • Cost: $5 per 1,000 requests (~$25–50/month for 50+ assignments/month)
  • Benefit: Real driving distance instead of estimates
  • Implementation: Simple API call from admin dashboard

For now, the estimate formula is accurate enough for tax purposes and can be verified with spot-checks using Google Maps.

Reporting & Export

Dashboard View

  • Weekly Total: Miles and deduction amount for current week
  • Monthly Total: Cumulative miles and annual projection
  • Quarterly Total: Tax planning view
  • Year-to-Date Total: Rolling annual deduction amount

Export Formats

Export Type Format Use Case

Tax-Ready Report PDF Provide to CPA or accountant at year-end

Integration with Financial Reporting

Mileage totals automatically feed into the accounting dashboard:

  • Total miles × $0.70 = mileage deduction
  • Deduction included in P&L calculations

Audit Defense

Maintain these records indefinitely:

  • Daily mileage logs (linked to assignments)
  • Assignment records (with dates and scope)
  • Annual summaries for tax years

If IRS ever audits, you can show: "On 3/15/2026, I completed appraisal of 123 Main St, visited 4 comparable properties, drove 35 miles per GPS/trip planning." This documentation is IRS-acceptable.

Payment Workflow (Detailed)

The payment workflow is the critical path that turns work into revenue. It's designed to be frictionless for clients while ensuring payment before report release.

Payment Enforcement: Report Gate

Critical Rule: Reports CANNOT be released until payment is confirmed.

Implementation: In the admin dashboard, the "Release Report" button is disabled until the invoice is marked as "Paid." This prevents accidental release before payment.

Payment Timeline Expectations

  • Same-day completion: Client receives payment link within hours of report completion
  • Fast payment: Card payments process instantly; ACH transfers typically 1–2 business days
  • Report access: Client access granted within minutes of payment confirmation
  • Stripe settlement: Funds appear in your bank account within 1–2 business days

Fallback Options

If a client prefers not to use Stripe:

  • Zelle Transfer: Client can transfer directly to your bank. Admin notes manual payment confirmation, then releases report.
  • Check: For lenders or corporate clients. Admin waits for check to clear before releasing report.
  • ACH via Stripe: Client can enter bank details on Stripe checkout page (lower fees)

Overdue Payment Handling

If invoice remains unpaid after 7 days:

  1. Admin dashboard displays "Overdue" badge
  2. Keith receives email reminder
  3. Optional: Send automated friendly reminder email to client
  4. Report remains unreleased until payment received

Discount & Proration Rules

The system supports:

  • Fixed price: Standard $X per appraisal
  • Custom pricing: Rush fee (+$100), additional comps (+$25 each), etc.
  • Manual adjustments: Keith can adjust invoice amount before sending payment link (e.g., client discount)

Changes to invoice amount trigger new Stripe payment link; customer doesn't see stale links.

Security Architecture

Protecting client data and financial information is non-negotiable. The security model is designed to be robust without sacrificing simplicity.

Infrastructure Security

HTTPS Everywhere

  • Netlify: Auto-SSL certificate for public site
  • All endpoints: Force HTTPS, no unencrypted connections

DNS & Domain

  • Cloudflare (Free): Manage DNS, basic DDoS protection
  • CAA Records: Restrict certificate issuance (prevent unauthorized SSL certs)

Authentication & Access Control

Admin Dashboard

  • Single User: Keith's personal dashboard, password-protected
  • Strong Password: Minimum 12 characters, stored hashed in localStorage (MVP) or Supabase Auth (future)
  • Session Timeout: 30 minutes inactivity → automatic logout
  • No Multi-User Support (MVP): If future staff needs access, upgrade to Supabase Auth with role-based access control

Client Portal

  • Supabase Auth: Email + password login, JWT tokens
  • Row-Level Security (RLS): Each client can only access their own reports
  • Password Reset: Email-based reset link (expires after 1 hour)
  • Session Management: JWT tokens expire after 24 hours

Data Protection

File Storage

  • Supabase Storage: Encrypted at rest with AES-256
  • File Access: Only authenticated clients can download their reports
  • File Types Restricted: Accept PDF and DOCX only, block executables
  • File Size Limit: 25MB max per report (prevents abuse)

Database

  • Supabase PostgreSQL: Encrypted in transit and at rest
  • Automatic Backups: Daily snapshots (Supabase free tier includes 7-day backup history)
  • Row-Level Security: Policies prevent unauthorized data access

Sensitive Information

  • Never Store: Credit card numbers, bank routing numbers, SSNs
  • Stripe Handles Cards: Stripe's PCI-compliant system, not your servers
  • Admin Notes: Keep client notes minimal, no sensitive data in descriptions

Application Security

CORS Configuration

  • Allow Only: designerhomesre.com, subdomain.designerhomesre.com
  • Block: Cross-origin requests from other domains

Content Security Policy (CSP)

  • Headers: Restrict inline scripts, external sources
  • Prevent: XSS (cross-site scripting) attacks, injection attacks

API Security

  • Stripe API Keys: Keep secret keys in environment variables, never in client code
  • Webhook Verification: Verify Stripe webhook signatures before processing
  • Rate Limiting: Limit API requests to prevent brute-force attacks

Data Breach Response Plan

  1. Detect: Monitor logs for unusual activity, Supabase alerts for anomalies
  2. Contain: Disable compromised account, rotate API keys
  3. Notify: Contact affected clients within 24 hours (email, phone)
  4. Investigate: Review logs, determine scope of exposure

Regular Security Checks
