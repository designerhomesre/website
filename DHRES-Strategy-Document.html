3. Full Copy Direction

Strategic messaging for each page (detailed copy is developed during build phase):

HOME

Tone: Professional, trustworthy, approachable. Goal: Immediate clarity of services + call to action. Lead with speed, accuracy, expertise. Hero CTA: "Request a Quote" or "Get Started."

ABOUT

Tone: Personal, credible, established. Goal: Build trust through credentials and experience. Include Keith Manning Jr.'s background, certifications (MAI, CRE, etc.), years of experience, company mission. Story-driven.

APPRAISAL SERVICES

Tone: Informative, detailed, authoritative. Goal: Educate on service types. Service cards for Residential, Commercial, FHA, Divorce, Estate, Retrospective, Green Home, etc. Include process steps, typical timeline, typical cost ranges.

MASS APPRAISAL CONSULTING

Tone: B2B expert, data-driven, strategic. Goal: Position as consultant to municipalities/governments. Highlight ROI, methodology, case studies. CTA: "Schedule Consultation."

SERVICE AREA

Tone: Clear, factual. Goal: Show geographic coverage. Interactive map showing Durham, Wake, Orange, Chatham counties. List specific cities (Raleigh, Durham, Chapel Hill, Carrboro, etc.).

REVIEWS

Tone: Social proof, authentic. Goal: Display trust signals. Aggregate Google, Yelp, BBB, Angi reviews. Show star ratings, snippets, link to full reviews on platforms.

REQUEST A QUOTE / INTAKE

Tone: Efficient, reassuring. Goal: Minimize friction. Smart form with conditional logic. Service type selector, property address, client type (individual/attorney/realtor), timeline urgency. Instant confirmation email.

CLIENT PORTAL

Tone: Functional, clean. Goal: Real-time transparency. Headline: "Track Your Order." Show status (pending/in-progress/complete), download reports, view invoices, message support.

FAQ

Tone: Helpful, accessible. Goal: Reduce inquiries. Cover: "What is an appraisal?", "How long does it take?", "How much does it cost?", "What do you need from me?", "Are you licensed?", "What areas do you serve?", "Can I dispute an appraisal?"

CONTACT

Tone: Welcoming, accessible. Goal: Multiple contact paths. Phone, email (info@designerhomesre.com), address, hours, embedded Google Map, contact form.

Typography

Headings: "Inter" font (or Roboto/Poppins as fallback). Bold weights (600-700). Generous letter-spacing for premium feel.

Body: System sans-serif stack (-apple-system, BlinkMacSystemFont, Segoe UI, Roboto). Regular weight (400), line-height 1.6-1.7.

Accents: Gold (#C9A84C) for borders, buttons, section dividers. Creates premium, sophisticated look.

Key Design Principles

  • Generous Spacing: Ample white space between sections (40-70px). Premium feel comes from breathing room, not clutter.
  • Clean Components: Minimal borders, subtle shadows (0 2px 8px rgba). Card-based layouts for services, testimonials.
  • Strong Hierarchy: Large, confident headings. Clear visual distinction between sections. Navy for primary text, charcoal for secondary.
  • Subtle Animations: Smooth hover states (0.3s ease transitions). Fade-ins on scroll (optional). No jarring effects—keep it professional.
  • Mobile-First: Responsive grid layouts. Typography scales. Touch-friendly buttons (48px+ height).
  • Accessibility: High contrast navy/charcoal text on light backgrounds. WCAG AA compliant colors. Proper heading hierarchy.

5. Tech Stack Recommendation

Optimized for speed, affordability, and professional results. Total cost: $7-15/month

Why Rebuild from WordPress?

  • Performance: Static HTML/CSS loads 3-5x faster than WordPress database queries. Better Core Web Vitals for SEO.
  • Security: No plugin vulnerabilities, no WordPress/database attacks, no hosting compromises. Static files = zero attack surface.
  • Cost: Netlify free tier replaces WordPress hosting ($10-20/mo). Google Workspace ($7/mo) stays for email. Total: ~$7/month vs. $25-40/month current.
  • Design Control: Full creative control over layout, animations, responsiveness. No theme limitations.
  • SEO: No bloat, no redirects, clean code = better SEO rankings. Schema markup, fast load times, mobile-first = ranking boost.
  • Maintenance: No updates, no plugin conflicts. Once built, it just works. Minimal ongoing overhead.

6. Hosting + Domain Recommendations

Domain Strategy

Keep current domain: designerhomesre.com is established and has any existing SEO authority. No need to migrate unless renewing at GoDaddy becomes problematic.

Option A (Simple): Keep GoDaddy for domain registration (~$15/year). Point nameservers to Cloudflare (free DNS). Cloudflare handles DNS, caching, DDoS protection.

Option B (Optimal): Migrate domain to Cloudflare (~$0.30/year renewal vs. $15). Cloudflare owns DNS, single dashboard.

Hosting Strategy

Primary Recommendation: Netlify

  • Free tier covers current + future growth (100GB bandwidth, unlimited sites, unlimited builds).
  • Automatic SSL/TLS certificates (HTTPS). Renewal automatic, no management needed.
  • Global CDN ensures fast load times for visitors across NC and beyond.
  • Built-in form handling (Netlify Forms) for contact/quote submissions.
  • Deploy via Git (GitHub/GitLab/Bitbucket). Push code = auto-deploy. Zero downtime.
  • Serverless functions available free tier (good for future API endpoints).
  • Analytics dashboard included.

Alternative: Vercel

Vercel also offers free tier with similar benefits. Slightly more optimized for Next.js, but HTML/CSS/JS works fine. Comparable to Netlify in features and pricing.

DNS Management with Cloudflare (Recommended)

Why Cloudflare?

Free features: DNS management, automatic HTTPS, DDoS protection, caching layer, analytics. Minimal setup needed once nameservers are pointed.

Setup: Point domain nameservers to Cloudflare → Cloudflare creates CNAME record pointing to Netlify → Netlify hosts site.

Email DNS Records

If using Google Workspace email: Cloudflare DNS will handle MX records pointing to Google's servers. One-time setup during Google Workspace onboarding.

SSL/HTTPS

Netlify provides free SSL certificates (auto-renew). No additional cost or management. All traffic encrypted.

Workflow & Automation (Zapier Free Tier)

Connect Typeform → HubSpot → Gmail via Zapier. Max 5 free zaps/month. Suggested automations:

Zap 1: Form Submission → HubSpot Contact

Trigger: New Typeform submission. Action: Create/update HubSpot contact with name, email, phone, service type, property address, etc.

Zap 2: Form Submission → Confirmation Email

Trigger: New Typeform submission. Action: Send Gmail confirmation email from info@designerhomesre.com. Template: "Thanks for submitting. We'll review and contact you within 24 hours."

Zap 3: HubSpot Deal Closed → Review Request Email

Trigger: HubSpot deal marked "Closed" (appraisal completed). Action: Wait 7 days, then send Gmail with Google Review request link + Yelp/BBB links.

Email Templates (HubSpot)

Create templates in HubSpot for recurring emails:

  • Quote Confirmation: "Your appraisal order received. Here's what to expect..."
  • In Progress Notification: "Your appraisal is underway. Expected completion: [date]."
  • Completion Notification: "Your appraisal is ready! Access via client portal or download link."
  • Review Request: "We'd love your feedback. Leave a review on Google/Yelp/BBB."
  • Follow-up (no contact): "Haven't heard from you? Questions about your appraisal?"

Integration Dashboard

Once set up, workflow is: Lead fills Typeform → Zapier auto-creates HubSpot contact + sends confirmation email → Team receives notification in Slack/email → Team updates deal status in HubSpot → Automation emails triggered based on status changes.

8. Client Portal Architecture

Transparent, real-time order tracking for clients. Two-phase rollout.

Phase 1: MVP (Month 1)

Tech Stack: Supabase (auth + database + storage) + HTML/CSS/JS frontend. Deploy alongside main site on Netlify.

Client-Side Features

  • Login/Register: Email + password auth via Supabase. First-time: register with email. Subsequent: login with email/password.
  • Dashboard: List of client's orders with status (Pending → In Progress → Under Review → Complete → Delivered).
  • Order Details: Property address, service type, order date, expected completion, total cost.
  • Report Download: Once status = "Delivered," download link appears. PDF stored in Supabase Storage.
  • Status Timeline: Visual timeline showing progression through stages + timestamp for each.

Admin-Side Features

  • Order Management: View all orders. Update status (simple dropdown). Admin URL: /admin/dashboard (protected by admin password).
  • Report Upload: Drag-drop PDF to attach to specific order. Auto-stores in Supabase Storage.
  • Status Change Emails: When admin changes status, automatic email sent to client. Example: "Your appraisal is now in progress. Expected completion: [date]."
  • Client List: View all registered clients, contact info, order history.

Phase 2: Full Dashboard (Month 2-3)

Expand with:

  • Messaging Thread: Per-order chat. Client can ask questions → admin replies. Reduces email threads.
  • Invoice/Payment History: View invoices, payment status, download receipts.
  • Notifications: Push notifications on status change (if client enables).
  • Admin Notifications: Email/Slack alerts when new order submitted or message posted.

Security & Access Control

  • Clients see only their own orders (Supabase RLS policies enforce this).
  • Admin section protected by password hash (stored in .env). No usernames/passwords for simplicity in MVP.
  • All communication over HTTPS (Netlify SSL).
  • PDFs stored in Supabase Storage with restricted access (only owner can download).

Integration with HubSpot

Portal order status syncs with HubSpot deal status. When admin marks order complete in portal, Zapier can trigger HubSpot deal update → review request email sent automatically.

9. AI Automation Plan

Leverage AI to reduce manual work and improve customer experience. Phased approach.

Smart Intake Form (Immediate)

Typeform with Conditional Logic

Acts as "AI-lite" intake. Conditional branching based on responses:

Example flow:

  • Q1: "What service do you need?" (Residential / Commercial / Consulting)
  • IF Residential: show "Is this for a purchase, refinance, or divorce?" → branches to different property questions
  • IF Commercial: show "Building type? Square footage? Annual NOI?"
  • IF Consulting: show "Jurisdiction? Number of properties? Timeline?"

Result: More accurate data capture. HubSpot receives pre-categorized leads. Less follow-up questions needed.

Smart Lead Categorization (HubSpot)

Automatic Property Mapping

Zapier maps Typeform answers to HubSpot properties: Service Type → [Residential/Commercial/Consulting]. Urgency Level → [High/Medium/Low]. This auto-populates the CRM, saving manual data entry.

Auto-Confirmations (Zapier)

Instant Response on Form Submit

Form submitted → Zapier triggers Gmail confirmation within seconds. Client gets immediate reassurance that submission received. Sets expectation: "We'll review your request and contact you within 24 hours."

AI Reply Drafts (Future - ChatGPT API)

Conversation Summarization

Future (Phase 4-5): Use OpenAI API to summarize client inquiry from form/email → generate draft response for team to review/send. Saves time on routine inquiries.

Example: Client asks "How long for residential appraisal?" → ChatGPT drafts: "Residential appraisals typically 24-48 hours. Scheduling depends on current volume. Here's our timeline..." → Team reviews/sends.

Automated Review Requests (HubSpot Workflows)

7-Day Post-Delivery Email Sequence

Trigger: Order status = "Delivered" in portal/HubSpot.

Action: Wait 7 days. Send email with links to Google, Yelp, BBB review forms. Personalized with client name + property address.

Optional 2nd touch: If no review within 14 days, send follow-up: "We'd still love your feedback..."

Result: Automated review generation. No manual email sending. More reviews → higher rankings on Google/Yelp.

Smart Follow-ups (HubSpot Workflows)

Stale Lead Automation

Trigger: No contact in 7 days from form submission.

Action: Send email: "Hi [Name], Just checking in on your appraisal request for [Property]. Any questions I can answer?"

Result: Re-engages cold leads without manual outreach.

Implementation Timeline

  • Week 1: Typeform conditional logic setup. Zapier integrations (form → HubSpot, form → email).
  • Week 2-3: HubSpot workflows (review requests, stale lead follow-ups).
  • Month 2+: Consider ChatGPT API integration for reply drafts if volume warrants.

Technical SEO (On-Site)

Schema Markup

Add JSON-LD schema to pages:

  • LocalBusiness: Homepage. Company name, phone, email, address, service area (multi-city). Helps Google Knowledge Panel.
  • Service: Each service page. Service name, description, provider, area served.
  • BreadcrumbList: Navigation path. Improves click-through rate from search results.
  • FAQPage: FAQ page. Google shows answers directly in search results (featured snippets).

Performance & Core Web Vitals

Static HTML = instant performance. Netlify CDN = <100ms global load times. Google PageSpeed Insights will show near-perfect scores. This is a major ranking factor.

Mobile-First Indexing

Responsive design (built into CSS from start). Mobile version loads as fast as desktop. Google prioritizes mobile experience.

Sitemap & Robots.txt

Netlify auto-generates sitemap.xml. Create robots.txt to guide crawlers. Submit sitemap to Google Search Console for faster indexing.
